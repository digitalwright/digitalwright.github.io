DECIPHER is a software that can be used to maintain, analyze, and decipher large amounts of DNA sequence data. DECIPHER is designed to complement other tools that are part of BioConductor, a suite of packages for the R programming language.

Installation

To install DECIPHER, start the latest version of R and enter:
if (!requireNamespace("BiocManager", quietly=TRUE))
    install.packages("BiocManager")
BiocManager::install("DECIPHER")
To ensure installation of the latest version, check that R is up-to-date before installation. To begin using DECIPHER see the Tutorials page.
