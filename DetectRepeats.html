Detect Repeats

The DetectRepeats function can find tandem or interspersed repeats in gene, protein, or genome sequences, as described in:

Cho, S.-T. & Wright, E. S. (2025) "Accurate detection of tandem repeats exposes ubiquitous reuse of biological sequences." Nucleic Acids Research, doi:10.1093/nar/gkaf866.

For an in-depth tutorial on DetectRepeats, see the "Detecting Obscure Tandem Repeats in Sequences" vignette, available from the Documentation page.

How do I detect tandem repeats?

First it is necessary to install DECIPHER and load the library in R. Next, upload sequences and run DetectRepeats to identify any tandem repeats.
