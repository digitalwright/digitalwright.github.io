R Lessons - Overview

These lessons are designed to provide a brief introduction to using the R programming language for scientific computing. These lessons describe how to:
  1. Work with different R data types.
  2. Manipulate text and search for patterns.
  3. Program with functions that interact.
  4. Import and export different file formats.
  5. Plot high quality graphs with base R.
  6. Repeatedly apply code with loops and maps.
