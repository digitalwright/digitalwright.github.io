Tutorials

The Tutorials section includes resources for everyone from novice to advanced R programmers. Although some of DECIPHER's functionality is accessible via this web tools, many features require using the DECIPHER package in R.

Short examples of using DECIPHER for sequence alignment can be found on the Alignment page. For installation directions, please see the Downloads page.
